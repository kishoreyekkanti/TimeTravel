This application will help you to fetch the good visiting places around your specified vicinity. If you go to some intresting place and take some photo through the app, it will be directly uploaded to twitpic along with the lattitude and longitude from where you clicked this photo. When any one wants to see the intresting places with in the user specified range say 1 to 100 KM, then all the photos in the raidus specified will be downloaded to the user mobile. He also has an option to see the distance between his present location and the location where photo is taken.
